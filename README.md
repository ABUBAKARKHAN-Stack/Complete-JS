# Complete JavaScript Learning Repository 🚀

Welcome to the **Complete JavaScript** repository! This project serves as a comprehensive collection of JavaScript concepts, code snippets, notes, and practical projects. It is designed to take you from the very basics of JavaScript all the way to advanced topics like Object-Oriented Programming (OOP), Asynchronous JavaScript, and API handling.

## 📖 About This Repository

Whether you are a beginner looking to understand the fundamentals or an experienced developer brushing up on advanced concepts, this repository contains neatly categorized folders with source code and explanations for various JavaScript features.

## 📂 Folder Structure & Topics Covered

The repository is structured into specific modules focusing on different areas of JavaScript:

### 🔰 Basics & Fundamentals
- `js_01_basic` & `js_02_basics`: Core introductory concepts and beginner scripts.
- `datatypes`, `strings`, `Number`, `date&time`: In-depth look at JavaScript's built-in types and methods.
- `let,const,var` & `memories`: Variable and memory management (Stack vs. Heap).
- `array` & `obj1`: Data structures, iterations, and object manipulation.
- `tut1`: Initial tutorial files.

### ⚙️ Functions & Scope
- `Functions`: Function declarations, expressions, and arrow functions.
- `Global&Local_Scope`: Understanding block, functional, and global scope.
- `Lexical Scope & Closures`: Advanced scope concepts and closure mechanisms.

### 🔄 Iteration & Data Manipulation
- `Maps-Filter-Reduce`: High-order array methods for functional programming.

### ⏳ Asynchronous JavaScript
- `Promises`: Handling asynchronous operations, `.then()`, `.catch()`, and `async/await`.
- `api`: Fetching data, making HTTP requests, and handling responses.

### 🏗️ Advanced JS & OOP
- `Classes` & `Classes_and_OOP`: Object-Oriented Programming in JS, prototypes, inheritance, and classes.
- `Modules`: ES6 modules, import/export syntax.
- `Regular Expression`: Pattern matching and string manipulation.
- `JS_Uncomfy`: Complex and tricky parts of JavaScript.

### 💻 Practical Projects
- `projects`: Contains real-world applications and mini-projects built using vanilla JavaScript.
  - `Projects-Part-1`
  - `Mega-Project`

## 🚀 Getting Started

To run any of the code snippets or projects locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ABUBAKARKHAN-Stack/Complete-JS
   ```
2. **Navigate into the directory:**
   ```bash
   cd Complete-JS
   ```
3. **Run the files using Node.js:**
   Make sure you have [Node.js](https://nodejs.org/) installed, then execute any file:
   ```bash
   node index.js
   ```
   *(Replace `index.js` with the specific file path, e.g., `node js_01_basic/strings.js`)*

## 🛠️ Environment Context
This repository includes a `.devcontainer` configuration, making it easy to open the project in a pre-configured Docker container via VS Code.
